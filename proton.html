This page contains tutorials for running games with Valve's Proton compatibility layer.
All games are tested on openSUSE Tumbleweed with specs found here

Grand Theft Auto V


Go to Steam > Settings > Steam Play and tick "Enable Steam Play for all other titles"
Download the latest version of GloriousEggroll's custom version of Proton from this link

Create this directory: ~/.steam/root/compatibilitytools.d


This can be done with this command:

mkdir ~/.steam/root/compatibilitytools.d

Extract the contents of Proton-VERSION.tar.gz to ~/.steam/root/compatibilitytools.d
Restart Steam.


Right Click on Grand Theft Auto V, then go to Properties > Compatibility and "Force the use of a specific Steam Play compatibility tool"
Select the Proton-GE version you downloaded.


Go to Properties > General and paste this string into the "Launch Options" box.

PULSE_LATENCY_MSEC=60 %command% -vulkan -USEALLAVAILABLECORES -cpuLoadRebalancing -ignorepipelinecache gamemoderun %command%

Launch Grand Theft Auto V and proceed through the Rockstar Games Launcher installation.
Please note, Processing Vulkan shaders can take a long time.
The game will launch windowed. It is recommended that you set the game to Fullscreen mode, with DirectX 11 enabled, along with setting "Refresh Rate" to match your monitor and enabling VSync. This should eliminate most stuttering in game.


The game should now be fully playable on Linux. Please keep in mind that anticheat software sometimes detects Proton as "unauthorised third-party software", therefore only play GTA Online if you are willing to take the risk of a possible ban.


Kerbal Space Program


Go to Steam > Settings > Steam Play and tick "Enable Steam Play for all other titles"
Right Click on Kerbal Space Program and navigate to Properties > Compatibility and select the latest available version of Proton.

Launch Kerbal Space Program.

The game should now be fully playable on Linux.
